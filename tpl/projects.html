HeyTaxi

HeyTaxi is a replacement for the current taxi booking service that relies on calling the dispatcher center which then books the taxi for a passenger. It is built to allow passengers and taxi drivers to communicate directly, without any intermediaries. Web application is used by taxi companies while android application by passengers.
Technologies: Flight PHP, MySQL, JavaScript, HTML/CSS, Ajax, REST, Java
